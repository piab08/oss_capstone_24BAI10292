## Shell Scripts

### Script 1: System Identity Report

This script will display information such as user, kernel version, uptime, and date of the operating system.

### Script 2: FOSS Package Inspector

This script will check whether Python is installed or not and then show information about Python.

### Script 3: Disk and Permission Auditor

This script will check the directory structure and permissions of the disk.

### Script 4: Log File Analyzer

This script will read a log file and count the number of occurrences of a given keyword.

### Script 5: Open Source Manifesto Generator

This script will generate a personalized open-source manifesto based on user input.


## How to Run the Scripts

```

./script1.sh

chmod +x script3.sh

./script5.sh
```
